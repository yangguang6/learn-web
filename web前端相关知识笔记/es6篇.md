## 8、`Map` / `WeakMap`，`Set` / `WeakSet`
[ECMAScript 6 入门 - Set 和 Map 数据结构](https://es6.ruanyifeng.com/#docs/set-map)

`WeakMap` 和 `WeakSet` 都是“弱引用”，不会被记入垃圾回收机制。

### `Map` 和 `WeakMap` 有什么区别？

|         |   Map   |       WeakMap       |
|   ---   |   ---   |          ---        |
|   键名   | 任意类型 | 只接受对象（null除外）  |
|  可遍历  |    是   |         否           |

### `Set` 和 `WeakSet` 有什么区别？

|         |   Set   |       WeakSet       |
|   ---   |   ---   |          ---        |
|   值    | 任意类型 | 只接受对象（null除外）  |
|  可遍历  |    是   |         否           |
