# 1. cookie、 sessionStorage和localStorage
  * ## 区别：
    1. 容量（cookie存储最大4KB，localStorage存储最大5MB）
    2. 是否会携带到ajax中（cookie每次都会在请求中携带，localStorage存储在本地）
    3. API易用性（cookie需要自己封装API，localStorage可以直接通过setItem(key,value)/getItem(key)调用）
